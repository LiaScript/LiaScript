Sizing Icons

Our Web Fonts + CSS include some basic controls for sizing icons in the context of your page’s UI.

Relative sizing

Icons inherit the font-size of their parent container which allow them to match any text you might use with them. With the following classes, we can increase or decrease the size of icons relative to that inherited font-size.

					
<i class="<%= className %> <%= className %>-home <%= className %>-xs"></i>
<i class="<%= className %> <%= className %>-home <%= className %>-sm"></i>
<i class="<%= className %> <%= className %>-home <%= className %>-lg"></i>
<i class="<%= className %> <%= className %>-home <%= className %>-2x"></i>
<i class="<%= className %> <%= className %>-home <%= className %>-3x"></i>
<i class="<%= className %> <%= className %>-home <%= className %>-5x"></i>
<i class="<%= className %> <%= className %>-home <%= className %>-7x"></i>

Fixed Width Icons

Need to vertically align a series of icons, like in a list or navigation menu? We’ve included a fixed-width support styling for just those cases.

Add a class of <%= className %>-fw on the HTML element referencing your icon to set one or more icons to the same fixed width. This is great to use when varying icon widths (e.g. a tall but skinny icon atop a wide but short icon) would throw off vertical alignment. For clarity in the following example, we’ve added a background color on the icon so you can see the fixed width and also bumped up the font-size of the parent element.

<div><i class="<%= className %> <%= className %>-arrow-up <%= className %>-fw" style="background:DodgerBlue"></i> Arrow Up</div>

Rotating Icons

Sometimes you need to rotate, flip, or mirror an icon for it to work in your project or design. We’ve included some quick utilities to help with that.

To arbitrarily rotate and flip icons, use the <%= className %>-rotate-* and <%= className %>-flip-* classes when you reference an icon.

					
<div class="<%= className %>-4x">
	<i class="<%= className %> <%= className %>-home"></i>
	<i class="<%= className %> <%= className %>-home <%= className %>-rotate-90"></i>
	<i class="<%= className %> <%= className %>-home <%= className %>-rotate-180"></i>
	<i class="<%= className %> <%= className %>-home <%= className %>-rotate-270"></i>
	<i class="<%= className %> <%= className %>-home <%= className %>-flip-horizontal"></i>
	<i class="<%= className %> <%= className %>-home <%= className %>-flip-vertical"></i>
	<i class="<%= className %> <%= className %>-home <%= className %>-flip-both"></i>
</div>

Rotate Classes

Class Rotation Amount
fa-rotate-90 90°
fa-rotate-180 180°
fa-rotate-270 270°
fa-flip-horizontal mirrors icon horizontally
fa-flip-vertical mirrors icon vertically
fa-flip-both mirrors icon vertically and horizontally (requires 5.7.0 or greater)

Animating

Need a loading or status communicating-icon to spin? You’ve come to the right place. We’ve included some basic animations in our support styling for you to use.

Use the <%= className %>-spin class to get any icon to rotate, and use <%= className %>-pulse to have it rotate with eight steps.

					
<i class="<%= className %> <%= className %>-home <%= className %>-3x <%= className %>-spin"></i>
<i class="<%= className %> <%= className %>-gear <%= className %>-3x <%= className %>-spin"></i>
<i class="<%= className %> <%= className %>-reload <%= className %>-3x <%= className %>-pulse"></i>
<i class="<%= className %> <%= className %>-arrow-left <%= className %>-3x <%= className %>-pulse"></i>

Stacked Icons

You can stack our icons easily with the support styling bundled with this font.

To stack multiple icons, use the <%= className %>-stack class on the parent HTML element of the 2 icons you want to stack. Then add the <%= className %>-stack-1x class for the regularly sized icon and add the <%= className %>-stack-2x class for the larger icon. <%= className %>-inverse can be added to the icon with the <%= className %>-stack-1x to help with a knock-out looking effect. You can even throw larger icon classes on the parent to get further control of sizing.

					
<span class="<%= className %>-stack <%= className %>-2x">
	<i class="<%= className %> <%= className %>-stack-2x <%= className %>-circle"></i>
	<i class="<%= className %> <%= className %>-stack-1x <%= className %>-twitter"></i>
</span>
<span class="<%= className %>-stack <%= className %>-2x">
	<i class="<%= className %> <%= className %>-stack-2x <%= className %>-ban"></i>
	<i class="<%= className %> <%= className %>-stack-1x <%= className %>-facebook"></i>
</span>

Aligning Stacked and Regular Icons

You can use stacked icons right alongside single icons. Since stacked icons are styled by our framework’s included CSS to be twice the size of single icons, you’ll need to either size down the stacked icon or size up the single icons.

					
<i class="<%= className %> <%= className %>-circle <%= className %>-2x"></i>
<span class="<%= className %>-stack" style="vertical-align: top;">
	<i class="<%= className %> <%= className %>-circle <%= className %>-stack-2x"></i>
	<i class="<%= className %> <%= className %>-twitter <%= className %>-stack-1x"></i>
</span>
<span class="<%= className %>-stack" style="vertical-align: top;">
	<i class="<%= className %> <%= className %>-circle-solid <%= className %>-stack-2x"></i>
	<i class="<%= className %> <%= className %>-twitter <%= className %>-stack-1x <%= className %>-inverse"></i>
</span>
<i class="<%= className %> <%= className %>-circle <%= className %>-2x"></i>
